# abalone

Classic game Abalone - play with yourself or over the network.

![Screenshot](screenshot.png)

## Usage

### Singleplayer game
```
java -jar Abalone.jar
```

### Network game - server
```
java -jar Abalone.jar -s <PORT>
```

### Network game - client
```
java -jar Abalone.jar -c <ADDRESS> <PORT>
```
